abTEM - ab initio Transmission Electron Microscopy in Python

abTEM is a flexible open-source package for simulating Transmission Electron Microscopy experiments. The Python API allows for easy integration with other simulation codes and analysis tools accessible from Python, while utilization of other open-source projects makes abTEM fast and scalable.
